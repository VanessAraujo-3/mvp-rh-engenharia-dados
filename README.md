# MVP Pipeline de Dados - Gestão de Pessoas

Este repositório contém o projeto de pós-graduação em **Ciência de Dados e Analytics** realizado por **Vanessa Araújo da Silva**.

O trabalho foi organizado em **8 notebooks**, correspondentes a cada etapa do pipeline de dados, desde a ingestão até a análise final. Além dos notebooks, também está disponível a **parte escrita em PDF**, contendo a explicação detalhada do trabalho, e os **dados em CSV** utilizados para análise.

## Estrutura do Repositório

- **notebooks/**
  - Notebook 1 – Resetar ambiente
  - Notebook 2 – Criação da camada **Bronze**
  - Notebook 3 – criação da camada **Silver**
  - Notebook 4 – criação da camada **Gold**
  - Notebook 5 – ETL Bronze
  - Notebook 6 – ETL Silver
  - Notebook 7 – ETL Gold
  - Notebook 8 – métricas e gráficos para análises

- **docs/**
  - Trabalho_escrito.pdf – Parte escrita do projeto

- **data/**
  - rh_dataset.csv – Dados de Recursos Humanos utilizados no projeto
